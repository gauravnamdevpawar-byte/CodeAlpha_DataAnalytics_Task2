# E-Commerce Sales Data Visualization Project

## Overview

This project focuses on transforming raw E-commerce sales data into meaningful and visually compelling insights using **Data Visualization techniques**.

The project uses:

* **Matplotlib**
* **Seaborn**
* Python-based visualization techniques

to analyze:

* sales trends,
* customer behavior,
* revenue patterns,
* category performance,
* regional sales,
* and business insights.

The visualizations are designed to:

* improve understanding,
* support decision-making,
* reveal hidden patterns,
* and create portfolio-quality analytical dashboards.

---

# Objectives

The main objectives of this project are:

* Transform raw data into meaningful visual formats
* Identify trends and patterns in sales
* Analyze customer purchasing behavior
* Detect anomalies and outliers
* Create professional and impactful visualizations
* Build a strong portfolio project for Data Analytics

---

# Dataset Information

The dataset contains E-commerce sales records with the following columns:

| Column Name      | Description                  |
| ---------------- | ---------------------------- |
| order_id         | Unique order ID              |
| order_date       | Date of order                |
| customer_id      | Customer identifier          |
| product_category | Product category             |
| region           | Sales region                 |
| quantity         | Quantity purchased           |
| unit_price       | Product price                |
| discount         | Discount percentage          |
| payment_method   | Payment mode                 |
| delivery_days    | Delivery duration            |
| customer_rating  | Customer satisfaction rating |
| revenue          | Total revenue generated      |

# Key Business Insights

## Major Findings

### 1. Electronics generates highest revenue

Electronics is the top-performing category.

### 2. Revenue strongly depends on:

* Unit Price
* Quantity Purchased

### 3. Discounts show weak negative impact

Heavy discounts do not significantly improve revenue.

### 4. Monthly trends reveal seasonal demand

Certain months generate significantly higher sales.

### 5. Customer ratings have weak relationship with revenue

Higher ratings do not strongly influence spending.

### 6. Revenue contains outliers

Some unusually high-value transactions exist.
